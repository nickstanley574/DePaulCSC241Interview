# Iterations with Indexes

```python

vehicles = ['sedan','train','truck','plane']

# Iterating thought the items
for v in vehicles:
    print(v)
print('Done.')


# Iterating though the indexs
for i in range(len(vehicles)):
    print(vehicles[i])
print('Done.')

```
